# Task Manager System

A Java-based task management application built with custom data structure implementations for task prioritization, dependency tracking, and analytics.

## Data Structures

### Custom (built from scratch)
- Singly Linked List — task history
- Doubly Linked List — bidirectional category navigation
- Circular Linked List — recurring tasks
- Stack — undo/redo functionality
- Queue — task scheduling
- Binary Search Tree — organize tasks by due date
- Graph — task dependency management with DFS/BFS

### Built-in (Java standard library)
- PriorityQueue — urgency-based scheduling
- HashSet — unique user IDs
- HashMap — user-to-task mapping

## Project Structure
src/
